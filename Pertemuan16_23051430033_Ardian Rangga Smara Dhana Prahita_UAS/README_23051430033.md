## Deskripsi
Aplikasi manajemen gudang berbasis React Native + Expo yang memiliki fitur:

- Inisialisasi & Arsitektur Dasar Sistem
- Implementasi Logika inventori dan segala macam stok
- Antarmuka Pengguna Utama (UI/UX) - Web & Mobile
- Manajemen Error & Stabilitas

## Cara Menjalankan Project

1. Extract file .zip

2. Masuk ke folder project

3. Install dependency

```
npm install
```

4. Jalankan project

```
npx expo start
```

5. Pilih platform:
   - Scan QR code menggunakan aplikasi **Expo Go** di perangkat fisik
